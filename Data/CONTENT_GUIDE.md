# Content Guidelines for Portfolio Website

## Profile Image
- **Location**: `Data/images/profile.jpg`
- **Size**: 400x400 pixels
- **Format**: JPG or PNG
- **Quality**: High resolution, professional photo
- **Background**: Clean, neutral background preferred

## Project Images
Add your project screenshots/images here with the following naming convention:
- `project1-main.jpg` - Main project image
- `project1-demo.gif` - Demo/animation if available
- `project2-main.jpg` - Second project image

**Guidelines:**
- Size: 800x600 pixels (16:12 ratio)
- Format: JPG for photos, PNG for screenshots
- Quality: Optimize for web (keep under 500KB each)

## Resume and Documents
- **Resume**: `Data/documents/Raghavendra_Thummala_Resume.pdf`
- **Certificates**: `Data/documents/certificates/`
- **Portfolio**: `Data/documents/portfolio/`

## Content to Update in HTML

### 1. Contact Information (index.html)
Update these placeholders:
- Email: your.email@example.com
- LinkedIn: linkedin.com/in/yourprofile
- GitHub: github.com/yourprofile
- Phone: +1 (XXX) XXX-XXXX

### 2. About Me Section
Replace "Content will be added here..." with:
- Professional summary (2-3 paragraphs)
- Career highlights
- Personal interests (optional)

### 3. Projects Section
For each project, include:
- Project title
- Brief description (2-3 sentences)
- Technologies used
- Link to demo/GitHub
- Key achievements

### 4. Blog/Articles Section
- Article titles
- Publication dates
- Brief excerpts
- Categories/tags
- External links

### 5. Media Section
- Video titles
- Platform (YouTube, Vimeo, etc.)
- Descriptions
- Embed links
